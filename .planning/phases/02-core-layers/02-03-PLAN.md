# Plan 02-03: ALPR Cameras (Supercluster) + Speed Cameras (Overpass)

## Goal
Display 336K+ ALPR/Flock cameras clustered with Supercluster and speed cameras from OSM Overpass API on the globe.

## Tasks

### Task 1: Overpass Speed Camera Fetcher
**Files:** `backend/src/fetchers/overpass.ts` (new)

Extend BaseFetcher:
- `sourceId: 'speed_cameras'`
- `defaultInterval: '0 0 */6 * * *'` (every 6 hours — static data)
- `cacheTTL: 21600` (6 hours)
- `fetchRaw()`: POST to `https://overpass-api.de/api/interpreter` with body:
  ```
  [out:json][timeout:120];node["highway"="speed_camera"];out body;
  ```
- `normalize()`: Convert Overpass elements to GeoJSON features
  - Each element: `{ type: "Feature", geometry: { type: "Point", coordinates: [lon, lat] }, properties: { id, maxspeed, direction, ref, type: "speed_camera" } }`
- Fallback to mirror `https://lz4.overpass-api.de/api/interpreter` on failure

### Task 2: ALPR/DeFlock Data Loader
**Files:** `backend/src/fetchers/deflock.ts` (new)

This is a STATIC dataset — load once, cache indefinitely:
- `sourceId: 'alpr'`
- `defaultInterval: '0 0 0 * * *'` (daily check)
- `cacheTTL: 86400` (24 hours)
- Data source strategy:
  1. Primary: Query Overpass API for `node["man_made"="surveillance"]["surveillance:type"="ALPR"]`
  2. This returns DeFlock-contributed ALPR cameras from OpenStreetMap
  3. Normalize to GeoJSON with properties: id, type, network info
- The full 336K FLOCK dataset is too large for Overpass (timeout). Use Overpass for what's available (~50-100K), and note that the full dataset requires the GitHub repo clone as a future enhancement.

### Task 5: Supercluster Clustering Hook
**Files:** `frontend/src/hooks/useCluster.ts` (new)

Create a hook that:
1. Takes GeoJSON features and returns clusters for the current viewport
2. Creates Supercluster index when data loads
3. Recalculates clusters when camera moves (debounced 200ms)
4. Uses `viewer.camera.computeViewRectangle()` for current bbox
5. Maps CesiumJS camera height to Supercluster zoom level (0-20)
6. Returns: `{ clusters, isLoading }`

### Task 6: SpeedCameraLayer Component

4. Click shows: speed limit, camera type, direction
