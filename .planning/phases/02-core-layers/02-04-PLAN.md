# Plan 02-04: Satellite Tracking with satellite.js

## Goal
Display satellite orbital positions on the globe with real-time client-side propagation using CelesTrak TLE data and satellite.js.

## Tasks

### Task 1: CelesTrak TLE Fetcher
**Files:** `backend/src/fetchers/celestrak.ts` (new)

Extend BaseFetcher:
- `sourceId: 'satellites'`
- `defaultInterval: '0 0 */4 * * *'` (every 4 hours — TLEs don't change fast)
- `cacheTTL: 14400` (4 hours)
- `fetchRaw()`: GET `https://celestrak.org/NORAD/elements/gp.php?GROUP=VISUAL&FORMAT=JSON`
  - Use VISUAL group for v1 (~200 bright satellites including ISS)
  - Returns OMM (Orbit Mean-elements Message) JSON
- `normalize()`: Convert OMM records to GeoJSON features
  - Each feature properties: OBJECT_NAME, NORAD_CAT_ID, EPOCH, MEAN_MOTION, ECCENTRICITY, INCLINATION, RA_OF_ASC_NODE, ARG_OF_PERICENTER, MEAN_ANOMALY, BSTAR, MEAN_MOTION_DOT, MEAN_MOTION_DDOT
  - Geometry: compute current position using satellite.js in backend (snapshot for initial load)
  - Include raw TLE-equivalent fields so frontend can propagate independently

### Task 4: SatelliteLayer Frontend Component
**Files:** `frontend/src/components/SatelliteLayer.tsx` (new)

This layer is UNIQUE — positions are computed CLIENT-SIDE at 60fps:
1. Get TLE/OMM data from `useLayerData('satellites')`
2. On data load: create `satrec` objects for each satellite using `twoline2satrec` or OMM conversion
3. Use `requestAnimationFrame` loop (or CesiumJS `scene.preRender` event) to:
   - Call `propagate(satrec, new Date())` for each satellite
   - Convert ECI to geodetic via `eciToGeodetic` + `gstime`
   - Update billboard positions in-place (no removeAll/re-add)
4. Render as BillboardCollection with satellite icon
5. Optionally draw orbital path as PolylineCollection (propagate +/- 45 minutes)
6. Click shows: name, NORAD ID, altitude, speed, inclination
7. Return `null`

### Task 6: Orbital Path Polylines
**Files:** `frontend/src/utils/orbitPath.ts` (new)

Utility to compute orbital path:
1. Given a satrec, propagate positions at 1-minute intervals for +/- 45 minutes (90 points)
2. Convert each to Cartesian3
3. Return array for PolylineCollection
4. Only compute for the currently selected/hovered satellite (not all 200)
